# asm-rs

ObjectWeb ASM implementation in Rust

## Getting started

Execute this inside your project!

```shell
cargo add asm-rs
```

Or manually add it in Cargo.toml

```toml
asm-rs = "<version>"
```

## Feature

- 1:1 (almost) ASM API
- Frame compute support
